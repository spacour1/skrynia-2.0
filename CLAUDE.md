# Claude Code Notes

Use `AGENTS.md` as the source of truth for project rules. It contains the hard
constraints for money, ledger immutability, payment idempotency, schema changes,
secrets, checks, and domain entrypoints.

For fast orientation, read `docs/agent-map.md` before opening large route/page
files. Avoid scanning generated or dependency folders: `node_modules`, `.next`,
`dist`, `uploads`, coverage output, and lockfiles unless the task specifically
requires them.
